# Predator Intel Undervolt

Script automático para configurar undervolt em notebooks Acer Predator Helios 300 usando Linux.

## Hardware testado

- Acer Predator Helios 300 G3-572
- Intel Core i7-7700HQ
- NVIDIA GTX 1060
- Parrot OS / Debian Linux

## Ajustes aplicados

| Componente | Undervolt |
|---|---:|
| CPU | -120 mV |
| CPU Cache | -120 mV |
| GPU Intel | -50 mV |
| System Agent | 0 mV |
| Analog I/O | 0 mV |
